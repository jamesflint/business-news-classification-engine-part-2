This classification engine was built by James Flint in 2018. It was a Capstone Project
for the Springboard Data Science Career Track course.

The engine is powered by a Long Short-Term Memory (LSTM) neural network,
trained on a proprietary business news dataset of summarised news articles provided
by Curation Corporation.

Source code for the model training and for this API, along with code for
models based on Convolutional Neural Nets (CNNs) and Very Deep Convolutional
Neural Nets (VDCNN) is available on Github.
